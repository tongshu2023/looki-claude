# 示例 · 薄荷(金毛)精选短视频

**背景**:Looki 30g 可穿戴相机天然是宠物互动视角 —— 你低头看狗的瞬间,它全在拍。

**目标**:让 Claude 把所有"和金毛互动"的片段精选出来,配字幕,准备发抖音/小红书。

## 一句话 prompt

```
调 looki 搜"薄荷""金毛""狗"三个关键词各 20 条去重,
按时间正序排列,抽每条首帧,
看一遍后挑出最打动人的 10 条,
给出:
- 每条的 moment_id + 时间 + 推荐 caption(3-5 秒节奏)
- 整段统一的抖音文案(<50 字,温暖风)
- 推荐的配乐类型(纯音乐/温柔女声/陶笛?)
- 发布的最佳时段建议
```

## Claude 自动执行

1. `looki_client.py search 薄荷` + `金毛` + `狗`
2. 合并去重(按 moment_id)
3. 按 `start_time` 升序
4. 对每条 `moments/{id}/files?highlight=true&limit=3` 拉高光帧
5. `frames.py` 抽帧 + Read 看
6. 打分:有狗主角面部 + 光线好 + 动作自然 = 高分
7. 挑 top 10
8. 按抖音 5s/条 + 转场 + 配乐格式输出剪辑清单

## 进阶:一周一次的"薄荷周报"

加到 cron:每周日晚跑一次。Claude 挑 5-10 个片段,生成:

- 剪辑清单(时间戳精确到毫秒)
- 配乐推荐
- 抖音/小红书双版本文案
- 封面建议(从高光帧里选一张)

## 你只需做一件事

挑**标题**,然后点**发布**。剪辑工具对接(剪映/CapCut)下个迭代做。
