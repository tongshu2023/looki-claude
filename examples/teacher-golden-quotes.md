# 示例 · 教学金句捕手

**背景**:Looki 的 `USER_EVENT_ANALYSIS` 会把你讲课中说的关键句提炼出来。实测作者(少儿编程老师)一周能捕到 5-10 句高含金量原话,比如:

- "教学不只是知识的传递,更是一种演技"
- "编程是在复杂逻辑中寻找对称美和规律"
- "取余运算是一种精准控制数据范围的实用编程技巧"

这些金句是**教师 IP 的核心资产**。

**目标**:每周自动把金句汇总成一份可复用的素材库。

## 一句话 prompt

```
调 looki for_you/items 拿 type=USER_EVENT_ANALYSIS 的所有条目,
过滤出 content 里包含教学/编程/算法/课堂相关的,
输出一张《本周金句表》,每条包含:
- 原句(不改字)
- 发生的课堂场景(哪天哪节课,学生/年级)
- 可复用的 3 种场景(朋友圈/公众号/课程 slogan)
- 延展方向(可以展开成一篇推文、一节课、一个 hook)
```

## 输出示例

```markdown
# 本周金句库 · 2026-W17

## 1. "教学不只是知识的传递,更是一种演技"
- 出处:2026-04-02 少儿编程教学教研会议
- 可复用:
  - 朋友圈:配图拍手板书 + "对小朋友讲代码,得会加戏"
  - 公众号:可以展开成一篇《少儿编程老师的演技进化史》
  - 课程 slogan:"学会'演'的少儿编程老师,才是真老师"
- 延展:可以做一期"老师教演技"的训练营

## 2. "编程是在复杂逻辑中寻找对称美和规律"
...
```

## 扩展 · 训练营素材引擎

作者把这个 skill 用作**少儿编程老师训练营**的底层素材引擎:

- 教师戴 Looki 上一周真实课堂
- 本 skill 自动扒出 20-30 条教学金句 + 课堂片段
- 训练营学员看到的**不是虚构示范课**,而是**一线老师真实一天的演技、节奏、应变**
- 每期训练营配套"真实课堂语料库"(50+ 金句 + 10+ 抽帧画面)

比起市面上"模拟课堂演示",真实语料的冲击力高一个量级。

这套玩法对课程机构 / 老师 IP 孵化器 / 转行培训者都成立。本仓库只提供**原料工具**,具体产品形态留给使用者设计。
